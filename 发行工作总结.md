## 🔄 发行流程

1. **本地构建**
   - 运行`build.bat`（Windows）或`./build.sh`（Linux/macOS）
   - 可执行文件将生成在`release`目录中

2. **GitHub自动发布**
   - 创建新版本标签：`git tag v1.1.0`
   - 推送标签：`git push origin v1.1.0`
   - GitHub Actions自动构建并发布Release
